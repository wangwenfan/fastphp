# fastphp
php微框架
> **框架目录结构**
```
根目录
|_ _  application 应用目录
|   　 |_ _common  公共函数  
|  　  |_ _controllers 控制器
|  　  |_ _models 模型
|   　 |_ _views  视图 
|_ _ config 配置文件
|_ _ fastphp 框架目录
|　 |_ _ common 公共函数
| 　|_ _ component 组件
|　 |_ _Controller.class.php 控制器基类
| 　|_ _ Core.php 核心文件
|　 |_ _ fastphp.php 启动文件
| 　|_ _ Model.class.php 模型基类
|　 |_ _ Sql.class.php 
| 　|_ _ View.class.php 视图基类
|_ _ index.php 入口文件
|_ _ .htaccess Apache路由配置 
```
